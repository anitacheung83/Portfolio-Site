Airline Reservation System

The airline reservation system is a web application built with React, Express.js, Node.js, and MongoDB. It utilizes Aero API to retrieve real-time flight information, such as flight status and seat availability. The application enables users to search and book flights, manage reservations, and make online payments securely.

The system is designed to provide a responsive and high-performance user interface with the use of React and Node.js. The frontend of the application is built with React, which allows for creating interactive and reusable components. The backend of the application is built with Express.js and Node.js, which enables fast and scalable application development. MongoDB is used to store and retrieve data efficiently, providing a reliable and robust database for the application. The combination of these technologies ensures the application is efficient, reliable, and user-friendly.

League Report Bot

The notification system is designed to provide real-time updates for the League of Legends gameplay statistics. Automatic Puzzle Solver

This Python project is an automatic puzzle solver that can solve Sudoku, Word Ladder, and Expression Tree puzzles using algorithms such as queues, stacks, and recursion. The solver includes visualization using the Matplotlib library, which displays the results of each puzzle. This project is a great opportunity to develop programming skills, problem-solving abilities, and to learn how to visualize data using Matplotlib.

By implementing different algorithms, this project can solve various types of puzzles efficiently and effectively. The solver takes input from the user and uses the appropriate algorithm to solve the puzzle. The results are then displayed using the Matplotlib library, which provides a clear and visual representation of the solution. Through this project, you can gain valuable experience in programming and develop problem-solving skills while learning how to use Matplotlib to visualize data.
